# React Shop

[Переглянути демо онлайн](https://react-shop-zeta.vercel.app/)

## Опис

**React Shop** — це простий інтернет-магазин, побудований на **React**, **Vite** та **Tailwind CSS**, з інтерактивною **корзиною** та адаптивним макетом.  

Особливості:

- Плавний responsive layout для десктопів, планшетів і мобільних пристроїв  
- Можливість додавати товари у корзину  
- Підрахунок загальної вартості з автоматичною знижкою 10% при покупці більше 5 одиниць  

---

## Технології

- React + TypeScript  
- Vite  
- Tailwind CSS  
- React Context API (для корзини)  

---

## Запуск локально

1. Клонуй репозиторій:
git clone https://github.com/FruntRoman/react-shop.git
cd react-shop

Встанови залежності:
npm install

Запусти dev-сервер:
npm run dev

Проєкт буде доступний за адресою http://localhost:5173/ (за замовчуванням).

Створи production-білд:
npm run build
Локально перевірити білд:
npm run preview
